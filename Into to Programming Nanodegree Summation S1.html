What is HTML
HTML is the standard mark-up language for creating web pages.
HTML consists of:
  • Text Content
  • Mark Up
  • Ref to Other Documents
  • Links
HTML uses mark ups to define contents these are called tags, these also help in defining white-space and how that space is viewed.
This is referred to as Mark Up

<tag name>Contents </tag name>
This forms an element

Element Types = In-line & Block

Some examples of tags and their uses:
<a> anchor in-line
<br> line break In-line Void tag requires no close
<p> paragraph Block
<img src="url" alt = "text"> image tag In-line

Lesson 2: Creating a Structured Document with HTML

The way a page is built is in a logic structure.
This is much like tree from roots to leaves.where the web page is the root and individual sections grows from that.
The <div> tag and <table> tag are ways to divide up a page into sections.
Developer Tools (in browsers) can be used to view code and structure of a HTML document or web page.

HTML the standard mark-up language used to create web pages.
CSS style sheet language used for describing the look and formatting of a document written in a mark-up language.
DOM a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other mark-up languages). The nodes of every document are organized in a tree structure, called the DOM tree.

Boxes
Boxes
Boxes

A web page can be divided into lots of boxes. Any thing or shape on a page is surrounded by a box. Smaller boxes fit into large and larger box until you reach the root of the tree being the web page.
The structure of the page would follow these defined boxes and be divided into sections of code, usually achieved with the use of the <div> tag. In the code these sections are indented to show nested sections.

Lesson 3: Adding CSS style to HTML Structure

When coding it is useful to reuse code as much as possiable.
The advantage is that there is less chance of an error occuring if you enter the code once instead of mulitply times.
Correcting or finding errors is easier because the code is in one place.
CSS is a good example as the "style" or code can be reused in many different parts of a HTML document.

Don't repeat yourself

Cascading refers to the inheritance of the rule or style with the most specific rule been applied.
CSS uses selectors to define a style. This can be a tag type or a defined class.
Example p{ } or .description{ }. The attributes are then defined inside the brackets.
A declaration is property and has a set value.

CSS can be linked into a HTML document in 3 ways:
